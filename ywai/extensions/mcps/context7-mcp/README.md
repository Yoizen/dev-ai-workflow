# context7-mcp

Instala una configuración de ejemplo de Context7 MCP dentro del proyecto:

- crea `.ywai/mcp/context7-mcp.example.json`

Esto sirve como referencia local para la capa de extensiones del installer.

> Nota: la configuración principal de Context7 ya existe a nivel global en:
>
> - `ywai/skills/setup.sh`
> - `ywai/config/opencode.json`

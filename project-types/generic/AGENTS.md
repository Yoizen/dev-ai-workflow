# Engineering Constitution

## Stack
Generic / language-agnostic. Use skills below for language-specific rules.

## Critical Rules
- No hardcoded secrets (env vars only)
- No god objects / single-responsibility
- Files ≤ 400 lines, functions ≤ 60 lines
- Tests for every new feature

# .NET Engineering Constitution

## Stack
- .NET 8/9, C#, ASP.NET Core
- Clean Architecture / CQRS (optional)

## Critical Rules
- Use `record` for DTOs and immutable data
- Nullable reference types enabled
- No exceptions for flow control (use Result pattern)
- Dependency injection via constructor
- Files ≤ 400 lines, methods ≤ 60 lines
